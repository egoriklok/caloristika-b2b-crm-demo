# PRD: Caloristika B2B CRM Demo

## Goal

Create a sales-ready demo page and local CRM demo package for a meeting with Caloristika B2B.

The pitch is not "another CRM". Caloristika already presents a personal account publicly. The demo must sell a vertical sales system:

- public B2B lead research,
- lead scoring by segment,
- tasting and pilot control,
- SKU matrix management,
- repeat-order control,
- Telegram catalog/order surface,
- AI-agent tasks for manager follow-up.

## Target Buyer

Development, sales, operations or founder-level stakeholder at Caloristika B2B.

## Demo Scope

- Prospect-specific landing page hosted via GitHub Pages.
- Local SQLite demo database generated in `../lunch-up-crm/data/caloristika_demo_crm.sqlite`.
- CRM runs from the same Lunch-Up CRM codebase using `LUNCH_UP_CRM_DB_PATH`.

## Required Evidence

- Public Caloristika B2B site: ready meals for coffee shops, vending, apart-hotels, fitness clubs and leisure centers.
- Public contact page: phone, email and Saint Petersburg location.
- Local CRM evidence: generated demo DB with products, leads, orders and pipeline stages.

## Non-Goals

- No scraping of private systems.
- No claim that Caloristika has approved the demo.
- No production deployment of the full CRM from this repo.
